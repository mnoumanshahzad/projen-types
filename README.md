# Projen Types

Additional [projen](https://github.com/projen/projen.git) project types

## Getting Started

To create a new project, run the following command and follow the instructions:

```bash
  $ mkdir my-project
  $ cd my-project
  $ npx projen new --from projen-types <PROJECT-TYPE>
```
